# 📋 Minha Lista de Tarefas

Aplicação web desenvolvida com ASP.NET Core MVC para gerenciar tarefas diárias.

##  Tecnologias
- C#
- ASP.NET Core
- Entity Framework Core
- Bootstrap

##  Funcionalidades
- ✅ Criar tarefas
- ✅ Marcar como concluídas
- ✅ Excluir tarefas
- ✅ Listar todas as tarefas

##  Como executar
1. Clone o repositório
2. Execute `dotnet run`
3. Acesse `https://localhost:5001`
